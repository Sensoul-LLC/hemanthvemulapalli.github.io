Featured Project

SENSOUL.

A ceiling-mounted, privacy-first elderly distress detection device that uses multi-sensor fusion — no camera, no microphone, no cloud dependency. All processing happens on-device at the edge.

When the system detects distress, a weighted danger score triggers a cancel window before automatically contacting emergency services. Built end-to-end: from hardware and sensor integration through data pipelines, ML inference, real-time dashboards, and automated alerting.
